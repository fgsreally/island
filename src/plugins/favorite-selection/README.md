# Favorite Selection Plugin

## 功能概述
这是一个用于收藏文件和文本的插件。

## 修改记录
- **[2026-04-11]**
  - 调整了岛屿全局尺寸（从 2 倍缩小到 1.5 倍左右），让其在屏幕上看起来更协调。
  - 修复了设置页无法滚动的问题，并将其移出了岛屿，作为一个独立的页面（`/settings`）在屏幕中央全屏显示。
  - 修复了光雾背景的层级问题，现在光雾可以正确显示在岛屿背景上，而不是在阴影层。
  - **关于“选中内容出现收藏区”的说明**：
    当前 Tauri 并不原生支持全局监听系统级别的文本选中事件（这通常需要编写特定平台的底层 Hook，如 Windows 的 UI Automation 或 macOS 的 Accessibility API）。
    目前的插件实现是通过监听 `tauri://file-drop` 等事件，支持将文件**拖拽到岛屿窗口上**进行收藏。
    如果您希望实现“在任意软件中选中文本，旁边出现一个悬浮的收藏区”，这需要：
    1. 开发一个后台进程（Rust 侧），使用系统 API 轮询剪贴板或监听全局鼠标/键盘事件来判断是否发生了“选中”。
    2. 获取当前鼠标的屏幕坐标。
    3. 动态创建一个新的无边框、透明的 Tauri 窗口，移动到该坐标附近显示“收藏区”。
    4. 处理拖拽或点击事件，将内容保存到数据库。
    这是一个相对庞大且需要深入操作系统底层 API 的功能，目前的插件架构暂未包含这部分底层实现。
